# HYRZ手游脚本
基础步骤类
1. 该步骤是否需要被保存
2. 该步骤的名称
3. 该步骤延迟执行时间设置（ms）
4. 该步骤执行方式：单次执行，多次执行，持续执行
5. 步骤本身可以被删除以及单步执行
6. 虚函数：待实现的步骤功能函数

程序启动步骤类（继承基础步骤类）
1. 程序运行设置：程序路径
2. 虚函数：执行程序启动

文本输入步骤类（继承基础步骤类）
1. 文本输入设置：填写输入文本项
2. 虚函数：执行文本粘贴（Ctrl+v）

按键输入步骤类（继承基础步骤类）
1. 按键录入设置：录入按键
2. 虚函数：模拟执行按键

图像识别步骤类（继承基础步骤类）
1. 图片识别设置：图片路径，是否支持模糊识别，识别区域设置（全屏还是部分）
2. 高级设置：识别精度设置，点击方式设置（左键单击，左键双击，右键单击），鼠标速度，点击偏移(X,Y)
3. 重写步骤执行方式中的多次执行和持续执行：增加选项：是否成功执行一次就停止，还是一直执行到次数或者时间耗尽
4. 虚函数：执行图像识别

窗口主体类
1. 提供界面布局
2. 支持添加步骤，支持步骤列表的导入导出
3. 支持全步骤执行，但步骤可以不启动从而跳过
4. 支持步骤的上下移动
5. 支持日志和执行状态显示
