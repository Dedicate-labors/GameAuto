# GUI思路
1. 需要具备如下功能：
  - 选择程序进行运行（判断调用是否成功）
  - 选择图片进行识别点击（判断调用是否成功）
  - 选择图片进行识别点击后，可以进行文本输入，即粘贴写好的文本（判断调用是否成功）
2. 上面的每个功能都可以定义一个多少ms后执行
3. 选择图片识别的功能时，可以提供是否支持模糊识别，以及选择预置和那个屏幕的那个区域进行识别等参数
4. 当功能被选择为多次执行时，还需要可配置参数：执行间隔ms，执行次数或者是持续执行时间（min或者s）
5. 每个功能可以视为一个步骤，我可以随意创建和组合
